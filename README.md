# Физалис · Матрица судьбы и натальная карта

Статический сайт-генератор двух эзотерических раскладов по дате рождения, в тёплом
кремово-персиковом стиле **«Физалис»**:

- **Матрица судьбы** — метод октаграммы (школа Ладини): дата раскладывается на 22
  Старших аркана, считаются личный и родовой квадраты, предназначение, кармический
  хвост, доминанта, возрастные декады и личные годы. Расчёт — порт `matrix_calc.py`
  из набора *physalis-decks-skill*, проверен на контрольных датах.
- **Натальная карта** — на реальных эфемеридах
  ([astronomy-engine](https://github.com/cosinekitty/astronomy)): положения 10 планет
  по знакам и домам, асцендент и MC, баланс стихий и аспекты. Точность сверена с
  Astrodienst до угловой минуты (карта А. Эйнштейна: Солнце 23°30′ Рыбы, Асц 11°38′ Рак).

Всё считается **в браузере** — бэкенда нет, данные никуда не отправляются.

## Запуск локально

Сайт статический. Достаточно открыть `index.html` в браузере.

Либо поднять локальный сервер (без Node/Python, на Windows):

```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1
# затем откройте http://localhost:8765/
```

## Структура

```
index.html              страница с двумя вкладками
css/style.css           дизайн-система «Физалис»
js/matrix.js            расчёт матрицы судьбы (порт matrix_calc.py)
js/octagram.js          SVG-октаграмма
js/natal.js             натальная карта на эфемеридах
js/cities.js            база городов (координаты + часовой пояс)
js/app.js               UI и рендер результатов
vendor/astronomy.browser.js   эфемериды (MIT, Don Cross)
assets/                 мотив-физалис
serve.ps1               локальный статический сервер
```
